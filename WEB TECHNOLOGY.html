Application
  • Applications is a set of Programs.
  • Programs are set of Instruction Which are given to a machines.
  • Programmers can write Instructions by using following type of languages:-
  1. Programming Language
  2. Query Language
  3. Markup Language
  4. Scripting Language
  • To run Application in to machine make use of Operating System like Windows, Linux, Unix, Mac, Android, Ios.
Types of Applications
  • We have four types of Applications are as follows:-
  1. Standalone Application
  2. Web Application
  3. Mobile Application
  4. Distributed Application
Standalone Application

  • The Applications which doesnt require Internet connectivity is reffered as Standalone Application.
  • To Develop this type of Application make use of "Core Java".
  • To run This Application, user need to install Operating System like Windows, Linux, Unix, Mac.

Server
  • Server is a machine which act as mediator between programmer and user.
  • Programmer are using server to store Web Applications.
  • User can access Web Applications by sending the request to the server with the help of internet and server will provide response to the user.
  • Server size / dimensions are depends on number of user data.

Note
  • Server and clinet machine both are present at different location.hence, web app required internet.
